# EasyPans - Full-Stack Recipe App

This is the code for the EasyPans startup, featuring a React frontend and a Node.js/Express backend.

## Tech Stack
- **Frontend:** React, Vite, TypeScript, Tailwind CSS, Shadcn/ui
- **Backend:** Node.js, Express, Mongoose
- **Database:** MongoDB Atlas

## How to Run

### Backend
1. Navigate to the `backend` folder.
2. Run `npm install`.
3. Create a `.env` file and add your `MONGO_URI`, `JWT_SECRET`, and `PORT`.
4. Run `npm run server`.

### Frontend
1. Navigate to the `frontend` folder.
2. Run `bun install` (or `npm install`).
3. Run `bun run dev` (or `npm run dev`).
